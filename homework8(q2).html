Web Page Evaluation

Homework #8 Q2

1) Appealing Website: Instagram

a) URL: https://www.instagram.com

b) Does this page use tables? No, Instagram does not use traditional HTML tables. The page uses CSS Grid and Flexbox for layout, which provides a responsive and modern design for organizing photos, stories, and feed content.

c) Does this page use CSS? Yes, it uses CSS extensively for page layout, text styling, colors, spacing, animations, and responsive design to work across different devices.

d) Is this page appealing or unappealing? This page is appealing to me.

  • The interface is clean and intuitive, making it easy to navigate and find content.
  • The visual design is attractive with good use of white space and high-quality image display.
  • The page loads quickly and responds smoothly to user interactions.

2) Unappealing Website: Netflix

a) URL: https://www.netflix.com

b) Does this page use tables? No, Netflix does not use HTML tables. It uses CSS-based layouts with Flexbox and Grid to create scrollable rows of content recommendations.

c) Does this page use CSS? Yes, it uses CSS for page layout, text styling, colors, dark theme aesthetics, and animations when hovering over content.

d) Is this page appealing or unappealing? For me, this page is unappealing in this comparison.

  • There is too much information displayed at once, making it overwhelming to browse.
  • The page feels crowded with endless rows of recommendations.
  • Important features are not easy to find quickly due to the cluttered layout.

e) If this page is unappealing, what would you do to improve it?

  • Reduce the number of recommendation rows on the homepage to make it less crowded.
  • Add more white space and better visual separation between content sections.
  • Make the search and filter options more prominent and easier to access.

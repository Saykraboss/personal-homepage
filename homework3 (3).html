Web Development Course — Debugging Exercises

Question 1

The background color is #0066CC (dark blue), so to ensure good contrast the text color must be white. The header block must use the Arial font family. The two missing HTML tags are the closing </style> and </head> tags.

Completed code:

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Trillium Media Design</title>
  <meta charset="utf-8">
  <style>
    body {
      background-color: #0066CC;
      color: white;
    }
    header {
      font-family: Arial, sans-serif;
    }
  </style>
</head>
<body>
  <header><h1>Trillium Media Design</h1></header>
  <p>Our professional staff takes pride in its working relationship
  with our clients by offering personalized services that listen to
  their needs, develop their target areas, and incorporate these
  items into a website that works.</p>
</body>
</html>

Summary of filled-in blanks:

• color: white; — white text provides strong contrast against the dark blue background.

• font-family: Arial, sans-serif; — sets the header font to Arial as required.

• </style> — closes the CSS style block.

• </head> — closes the head section before the body begins.

Question 2

The page will not display properly because the <style> block inside <head> is never closed correctly. On the line after the CSS rules, the code reads: 

  font-size: 1.2em; }
<style>       ← incorrect: opens a second style block instead of closing the first
</head>

It should be </style> (a closing tag), not <style> (an opening tag). Because the browser never sees </style>, it treats all subsequent HTML as raw CSS text, so the page body is either invisible or rendered without any styling.

Corrected line:

  font-size: 1.2em; }
</style>      ← correct closing tag
</head>
